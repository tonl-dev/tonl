Getting Started

Install and start using TONL in 30 seconds

Installation

npm install tonl

Quick Example

import { TONLDocument } from 'tonl';

// Create from JSON
const doc = TONLDocument.fromJSON({
  users: [{ name: 'Alice', age: 30 }]
});

// Query
const result = doc.query('users[*].name');

// Modify
doc.set('users[0].age', 31);

// Save
await doc.save('data.tonl');

API Reference

Core functions and TONLDocument class

TONLDocument.fromJSON()

Create document from JavaScript data

TONLDocument.fromJSON(data)

query()

Query data with JSONPath-like expressions

doc.query('users[?(@.age > 18)]')

set() / delete()

Modify data at any path

doc.set('user.age', 31).delete('user.temp')

Query Syntax

JSONPath-like query expressions

user.name
Property access
users[0]
Array indexing
users[*].name
Wildcard (all names)
$..email
Recursive descent (all emails at any depth)
users[?(@.age > 18)]
Filter expression

CLI Tools

Command-line interface for TONL

ENCODE
tonl encode data.json --smart
DECODE
tonl decode data.tonl
QUERY
tonl query users.tonl 'users[*]'
GET
tonl get data.tonl "user.name"
VALIDATE
tonl validate --schema schema.tonl
STATS
tonl stats data.json
