Quick Start

Get started in 30 seconds

1. Install
npm install tonl
2. Import and Use
import { TONLDocument } from 'tonl';

// Create document
const doc = TONLDocument.fromJSON({ users: [...] });

// Query data
doc.query('users[?(@.role == "admin")]');

// Modify and save
doc.set('users[0].age', 31);
await doc.save('data.tonl');

That's it!

You're now using TONL with full query, modification, and streaming capabilities.
